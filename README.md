**Вопрос:** Для чего нужны параметры `preemptible = true` и `core_fraction = 5`?

**Ответ:** 
- `preemptible = true` — прерываемая ВМ, стоимость в 3-4 раза дешевле, но может быть остановлена в любой момент. Подходит для обучения и тестирования.
- `core_fraction = 5` — гарантируется только 5% производительности ядра, что снижает стоимость. Достаточно для нетребовательных задач.

## Задание 6. Использование map(object) и metadata

### Финальный terraform plan (0 изменений):

![Задание 6](screenshots/Задание%206.png)

**Вопросы:**

1. **Для чего используется map(object) переменная?**
   - Позволяет группировать связанные параметры (cores, memory, core_fraction, hdd_size, hdd_type) в одной переменной. Удобно для описания конфигурации каждой ВМ.

2. **Почему metadata вынесена в отдельную переменную?**
   - Чтобы переиспользовать одинаковые настройки (ssh-keys, serial-port-enable) для всех ВМ и избежать дублирования кода.
